Release v0.2.2

  • New: introduces JavaScript API V3, to be used with GraalJS
  • New: adds support for ECMAScript Modules (ES Modules)

The JavaScript API V2 is deprecated.
End of 2022 it will be removed from the scripting plugin. Consider to migrate your scripts to the JavaScript API V3.

Support for Mozilla Rhino and the JavaScript API V1 is deprecated.
End of 2022 it will be removed from the scripting plugin. Consider to migrate your scripts to the GraalJS engine and the JavaScript API V2.
