Release v0.2.0

  • major cleanup
  • New: requires Java 11
  • New: supports GraalJS in JOSM running with GraalVM
  • New: supports GraalJS with a stock JDK
  • New: introduces JavaScript API V2, to be used with GraalJS

Support for Mozilla Rhino and the JavaScript API V1 is deprecated.
End of 2022 it will be removed from the scripting plugin. Consider to migrate your scripts to the GraalJS engine and the JavaScript API V2.
