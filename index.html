JOSM Scripting Plugin

Run scripts in the Open Street Map editor JOSM

The JOSM Scripting Plugin is a plugin for the Open Street Map editor JOSM.

It includes Oracles Javascript engine Graal.js and an API for Javascript to manipulate JOSMs internal application objects.

{% highlight javascript %} const command = require("josm/command") const { NodeBuilder } = require("josm/builder") // create an add command ... const cmd = command.add( NodeBuilder .withTags({amenity: "restaurant"}) .withPosition(12.34,45.67) .create() ) // ... and apply it to the active layer cmd.applyTo(josm.layers.activeLayer) josm.alert("Added a node") {% endhighlight %}

It also executes Python scripts and can load and execute plugins written in Python.

In addition, it can execute scripts written in Python, Groovy, Ruby, or any other language, for which a JSR-223 compatible script engine is available.
