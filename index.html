JOSM Scripting Plugin

Run scripts in the Open Street Map editor JOSM

The JOSM Scripting Plugin is a plugin for the Open Street Map editor JOSM.

It includes a built-in "native" scripting engine for Javascript based on Mozilla Rhino and an API for Javascript to manipulate JOSMs internal application objects.

{% highlight javascript %} var command = require("josm/command"); var nodeBuilder = require("josm/builder").NodeBuilder; josm.activeLayer.apply( command.add( nodeBuilder .withTags({amenity: "restaurant"}) .withPosition(12.34,45.67) .create()) ) ); josm.alert("Added a node"); {% endhighlight %}

In addition, it can execute scripts written in Groovy, Python, or Ruby.
