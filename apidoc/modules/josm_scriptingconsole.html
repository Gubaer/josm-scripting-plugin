Module josm/scriptingconsole

Provides a set of functions to write to the built-in scripting console.

Name Description
Methods
clear Clears the scripting console.
hide Hides the scripting console
print Prints a string to the console
println Prints a string to the console, including newline
show Shows the scripting console
toggle Toggles the visibility of the scripting console

Functions

clear

Clears the scripting console.

Examples

{% highlight javascript %} var console = require("josm/scriptingconsole"); console.clear(); {% endhighlight %}

hide

Hides the scripting console

Examples

{% highlight javascript %} var console = require("josm/scriptingconsole"); console.hide(); {% endhighlight %}

print

Prints a string to the console

Examples

{% highlight javascript %} var console = require("josm/scriptingconsole"); console.print("Hello world!"); // or use formatting console.print("Hello world! My name is {0}", "foo"); {% endhighlight %}

println

Prints a string to the console, including newline

Examples

{% highlight javascript %} var console = require("josm/scriptingconsole"); console.println("Hello world!"); // or use formatting console.println("Hello world! My name is {0}", "foo"); {% endhighlight %}

show

Shows the scripting console

Examples

{% highlight javascript %} var console = require("josm/scriptingconsole"); console.show(); {% endhighlight %}

toggle

Toggles the visibility of the scripting console

Examples

{% highlight javascript %} var console = require("josm/scriptingconsole"); console.toggle(); {% endhighlight %}
