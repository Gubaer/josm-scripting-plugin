Module josm/mixin/DataSetMixin

This module is auto-loaded by the scripting plugin and mixed into the native java class DataSet.

This module provides the following classes:

  • DataSetSelectionFacade
This module provides the following mixins:
  • DataSetMixin

Note, that mixins are loaded automatically by JOSM. You don't have to load them using require("my/mixin").
