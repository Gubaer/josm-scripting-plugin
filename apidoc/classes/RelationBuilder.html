Class RelationBuilder

RelationBuilder helps to create OSM Relations.

Methods of RelationBuilder can be used in a static and in an instance context. It isn't necessary to create an instance of RelationBuilder, unless it is configured with a DataSet, which created ways are added to.

{% highlight javascript %} var rbuilder = require("josm/builder").RelationBuilder; // create a new local relation var r1 = rbuilder.create(); // create a new global way var r2 = rbuilder.withTags({route: 'bicycle'}).create(123456); // create a new proxy for a global relation (an "incomplete" node in JOSM terminology) var r3 = rbuilder.createProxy(123456); {% endhighlight %}
