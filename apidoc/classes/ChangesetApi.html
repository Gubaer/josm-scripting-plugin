Class ChangesetApi

Provides methods to open, close, get, update, etc. changesets on the OSM API server.

Note: this class doesn't provide a constructor. Methods and properties are "static".

{% highlight javascript %} // load the changeset api var api = require("josm/api").ChangesetApi; // create a new changeset on the server var cs = api.open(); {% endhighlight %}
