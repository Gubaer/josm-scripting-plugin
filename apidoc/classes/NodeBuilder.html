Class NodeBuilder

NodeBuilder helps to create OSM nodes.

Methods of NodeBuilder can be used in a static and in an instance context. It isn't necessary to create an instance of NodeBuilder, unless it is configured with a DataSet, which created nodes are added to.

{% highlight javascript %} var nbuilder = require("josm/builder").NodeBuilder; // create a new local node at position (0,0) without tags var n1 = nbuilder.create(); // create a new global node at a specific position with tags var n2 = nbuilder.withPosition(1,1).withTags({name: 'test'}).create(123456); // create a new proxy for a global node (an "incomplete" node in JOSM terminology) var n3 = nbuilder.createProxy(123456); {% endhighlight %}
