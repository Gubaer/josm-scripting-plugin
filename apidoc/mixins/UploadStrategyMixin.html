Mixin UploadStrategyMixin

This mixin provides additional properties and methods which you can invoke on an instance of UploadStrategy.

This mixin provides additional properties and methods for the native Java class UploadStrategy, whose native public methods are available for scripting, too. If a native method name is hidden by property name in the mixin, prefix the native name with $.
