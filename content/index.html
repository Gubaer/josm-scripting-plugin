    • JOSM Scripting Plugin

    Run scripts in the Open Street Map editor JOSM.

    The JOSM Scripting Plugin is a plugin for the Open Street Map editor JOSM.

    It supports several scripting languages, including Groovy, Python, or Ruby. Any scripting language with JSR-223 compatible script engine can be used.

    The plugin includes a built-in "native" scripting engine for JavaScript based on Mozilla Rhino and an JavaScript API to manipulate JOSMs internal application objects.

    Install

    1. Download and install JOSM
    2. Open the Preferences dialog and select the plugins tab.
    3. Update the list of available plugins and search for scripting.

    4. Activate the plugin and restart JOSM.
