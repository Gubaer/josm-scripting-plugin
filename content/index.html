    Run scripts in the Open Street Map editor JOSM.

    • Tweet

    The JOSM Scripting Plugin is a plugin for the Open Street Map editor JOSM.

    It includes a built-in "native" scripting engine for Javascript based on Mozilla Rhino and an API for Javascript to manipulate JOSMs internal application objects.

    In addition, it supports several other scripting languages, including Groovy, Python, or Ruby.
