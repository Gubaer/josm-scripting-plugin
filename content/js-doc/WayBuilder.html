Methods

<static> create

Creates a new way.

Can be used in an instance or in a static context.

. Optional named arguments in the parameters args
  • id - the id of a global way (number > 0)
  • version - the version of a global way (number > 0)
  • nodes - an array or a list of nodes
  • tags - an object with tags. Null values and undefined values are ignored. Any other value is converted to a string. Leading and trailing white space in keys is removed.
Parameters:
Name Type Description
id number (optional) a global way id. If missing and not set before using withId(..), creates a new local id.
args object (optional) additional parameters for creating the way

<static> createProxy

Creates a new proxy way. A proxy way is a way, for which we only know its global id. In order to know more details (nodes, tags, etc.), we would have to download it from the OSM server.

The method can be used in a static and in an instance context.

<static> withNodes

Declares the nodes of the way.

Accepts either a vararg list of nodes, an array of nodes or a Java list of nodes. At least two non-identical nodes have to be supplied. The same node can occure more than once in the list, but a consecutive sequence of the same node is collapsed to one node.

The method can be used in a static and in an instance context.

Parameters:

nodes object... the list of nodes. See description and examples.
