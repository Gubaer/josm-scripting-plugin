LatLonMixin

LatLonMixin provides additional properties and methods which you can invoke on an instance of LatLon. The native methods of JOSM Node are still available for scripting. If they are masked by methods and properties defined in this wrapper, you can access them by adding the prefix charater $.

Example
var LatLon = org.openstreetmap.josm.data.coor.LatLon;

// set members 
var pos = new LatLon(1,2);
pos.lat;     // -> 1

// you can still invoke the native java method lat(), just add $ as prefix.
// Note the parentheses - the native method has to be called as function!
pos.$lat();  // -> 1
