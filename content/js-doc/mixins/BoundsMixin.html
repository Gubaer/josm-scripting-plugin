Mixin BoundsMixin

This mixin is auto-loaded by the scripting plugin and mixed into the native java class Bounds. It provides additional properties and methods which you can invoke on an instance of Bounds.

The native methods of Bounds are still available for scripting. Just prefix their name with $ if they are hidden by properties or functions defined in this mixin.
