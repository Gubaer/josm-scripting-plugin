NodeMixin

NodeMixin provides additional properties and methods which you can invoke on an instance of Node.

NodeMixin inherits the properties and methods of OsmPrimitiveMixin.

lat :number

Get or set the node latitude.

get: - replies the current latitude, or undefined, if the latitude isn't known.

set: - assign the latitude. Expects a number in the range [-90,90]. Raises an error, if the node is a proxy node.

pos :org.openstreetmap.josm.data.coor.LatLon

Get or set the node position.

get: - replies an instance of LatLon or undefined, if the position isn't known.

set: - assign the position. Either an instance of LatLon, or an object with the properties {lat: ..., lon: ...}

Source:
  • NodeMixin.js, line 145
Example
var LatLon = org.openstreetmap.josm.data.coor.LatLon;
var nb = require("josm/builder").NodeBuilder;

// assign a LatLon as position  
n.pos = new LatLon(23, 32.33);

// assign an object as position 
n.pos = {lat: 23, lon: 32.33};

// get the position
n.pos;  // -> a LatLon with the position
