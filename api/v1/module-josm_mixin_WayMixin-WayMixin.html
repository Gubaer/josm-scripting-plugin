Mixin: WayMixin

josm/mixin/WayMixin~ WayMixin

API V1 is deprecated and will be removed in 2021. Consider to migrate your scripts to API V2.
This mixin provides additional properties and methods which you can invoke on an instance of Way. You can access nodes using indexed properties, i.e. 
way[i]
, see example below.
