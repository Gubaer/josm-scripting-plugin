Mixin: UploadStrategyMixin

josm/mixin/UploadStrategyMixin~ UploadStrategyMixin

API V1 is deprecated and will be removed end of 2022. Consider to migrate your scripts to API V3.
This mixin provides additional properties and methods which you can invoke on an instance of UploadStrategy.

<static> from(value)

Convert value to UploadStrategy

Checks whether a value can be converted into an UploadStrategy, converts and normalizes it, and returns the converted value. Throws an error, if value can't be converted. Accepts one of the following values:
a UploadStrategy
just replies value
a string
One of the following strings:
  • individualobjects
  • chunked
  • singlerequest
Replies the matching enumeration value UploadStrategy
Throws an error, if value is null or undefined.
Parameters:
